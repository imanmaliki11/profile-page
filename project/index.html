DSOP

DSOP (Digital Services Orchestration Platform) is a web-based application for managing a WiFi ecosystem, this project was created with Laravel 7 framework and PHP 7 as a programming language. This project not only builds a web service but also should be integrated with the free radius framework and should manage many routers.

This Project belongs to PT Didanataneca as a service for managing community WiFi projects in several Villages in Kalimantan, this project can manage WiFi Users, Buy a WiFi plan with Xendit payment gateway, and a PPOB for buy and pay pulsa, paket data, PDAM, PLN, and other PPOB products.

Technology / Framework

Laravel 7, PHP 7, MySQL, AWS, Xendit Payment Gateway, Radiusdesk(freeradius), Ayoconnect PPOB, Bootstrap 4.5, and others.

Bakoel Omah (Klikbuy)

Bakoel Omah is an e-commerce application focused on selling food and beverage with the main idea is for helping UMKM to grow up, created with a mobile view and runs on a browser sellers and buyers don't need to install an application. This application also connects with social media for automation selling on Instagram, Facebook and other social media.

This Project belongs to PT Luoshive with a Business Client. This application was created with Laravel 8 as a main framework with PHP 8 as a programming language and connect with API (Facebook graph API) to connect with Instagram and Facebook and make a transaction easy for buyers.

Technology / Framework

Laravel 8, PHP 8, PostgreSQL, Instagram API, Meta API, Bootstrap 4.5, Figma, Docker, and otherS.

DNE Track

DNE Track is an IoT project for tracking a device on a blank internet area using Astrocast as a data transmitter (satellite), not only for tracking This project also wants to send an important data over to internet with the Astrocast satellite.

This Project belongs to PT Didanataneca as its own company product for helping to track any device on a blank internet area, we also provide send data based on our Client's request and calculate data to show proprtly on the dashboard.

Technology / Framework

Laravel 7, PHP 7, MySQL, Astrocast Node S+, Ardunio, Bootstrap 4.5 and others.
